# Security Policy

## Supported Versions

This project is in early development (pre-1.0). Only the latest published version on `@conductor-kit/*` receives fixes.

## Reporting a Vulnerability

Please **do not open a public issue** for security vulnerabilities.

Instead, report privately via [GitHub Security Advisories](https://github.com/chanshin0/conductor-kit/security/advisories/new).

Include:

- A description of the issue
- Steps to reproduce
- Affected packages and versions
- Any suggested mitigation

I'll acknowledge within 5 business days and aim to triage within 2 weeks. Credits are given in the release notes unless you prefer to remain anonymous.
